# Images d'exemple pour les Flashs Disponibles

Ce dossier contient les images d'exemple qui seront affichées dans la section "Flashs Disponibles" de la page publique de l'artiste.

## Images à ajouter

Ajoutez les images suivantes dans ce dossier avec les noms exacts :

### Rangée du haut :
1. **serpent-floral.jpg** - Tatouage de serpent et fleurs en style "fine line" noir et gris sur un avant-bras
2. **dague-old-school.jpg** - Tatouage traditionnel coloré d'une dague traversant une rose avec des hirondelles sur le haut d'un bras
3. **papillon-abstrait.jpg** - Tatouage de papillon mêlant style géométrique et aquarelle (bleu, violet) sur une omoplate

### Rangée du bas :
4. **crane-boussole.jpg** - Tatouage géométrique de crâne et boussole sur un mollet (marqué "VENDU" dans l'interface)
5. **rose-realiste.jpg** - Tatouage réaliste d'une rose rose épanouie sur une cuisse
6. **paysage-blackwork.jpg** - Tatouage "blackwork" d'un paysage de montagne et forêt dans un losange sur le haut du dos

## Format recommandé

- **Format** : JPG ou PNG
- **Taille** : 800x800px minimum (carré recommandé)
- **Poids** : < 500KB par image pour un chargement optimal
- **Style** : Noir et blanc, adapté pour le tatouage

## Utilisation

Ces images sont automatiquement affichées dans la section "Flashs Disponibles" de la page publique de l'artiste, en plus des flashs réels créés par l'artiste.

Les images sont définies dans `constants/flashExamples.ts` avec leurs métadonnées (titre, prix, durée, etc.).
